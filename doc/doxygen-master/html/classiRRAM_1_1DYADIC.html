DYADIC approx ( const REAL &  x,
const int  p 
)
friend

Computes a dyadic approximation to x that is accurate to 2^p.

This is a multi-valued function.

Reiteration is used to ensure \(x_\varepsilon\leq2^{p+1}\). The computed value at least as precise as \(x_c\) rounded to a multiple of \(2^{p-1}\).

Parameters
xthe real number to approximate
paccuracy
Returns
a dyadic approximation q that satisfies |x-q|<2^p.
Exceptions
Iterationwhen \(2^{p+1}<x_\varepsilon\)
See also
